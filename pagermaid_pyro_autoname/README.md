# autoname_plus\n\nPagerMaid-Pyro 自动改名增强插件。\n\n## 功能\n- 自动按时间更新 Telegram `last_name`\n- 支持启用/关闭\n- 支持自定义模板\n- 支持自定义 UTC 偏移\n- 支持预览和手动应用\n\n## 命令\n- `autoname on`：启用自动改名\n- `autoname off`：关闭自动改名\n- `autoname status`：查看当前状态\n- `autoname preview`：预览当前生成结果\n- `autoname apply`：立即应用一次\n- `autoname offset 8`：设置 UTC 偏移\n- `autoname set {HH}:{mm} {tz} {clock}`：设置模板\n\n## 模板变量\n- `{HH}`：24 小时\n- `{hh}`：12 小时\n- `{mm}`：分钟\n- `{ss}`：秒\n- `{ampm}`：AM / PM\n- `{weekday}`：周几\n- `{date}`：日期\n- `{time}`：时间\n- `{tz}`：时区文本，例如 `UTC+8`\n- `{offset}`：偏移，例如 `+8`\n- `{clock}`：时钟 emoji\n\n## 目录结构\n```\nautoname_plus/\n├── DES.md\n└── main.py\n```\n\n建议投放到 PagerMaid-Pyro 插件目录后加载使用。\n

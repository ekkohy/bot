# schedmsg_plus\n\nPagerMaid-Pyro 定时发消息插件（最终整理版）。\n\n## 功能\n- 定时向某个人发送消息\n- 定时向某个群 / 频道发送消息\n- 支持每日任务\n- 支持一次性任务\n- 支持启用、禁用、删除、立即发送\n- 支持自定义 UTC 偏移\n- 内置防重复发送逻辑（进程锁 + 发送前落库标记）\n\n## 命令\n- `schedmsg adddaily <名称>|<目标>|<HH:MM>|<消息>`\n- `schedmsg addonce <名称>|<目标>|<YYYY-MM-DD HH:MM>|<消息>`\n- `schedmsg list`\n- `schedmsg del <名称>`\n- `schedmsg on <名称>`\n- `schedmsg off <名称>`\n- `schedmsg run <名称>`\n- `schedmsg tz <UTC偏移>`\n\n## 示例\n### 每天早上 9 点给群发消息\n```\nschedmsg adddaily morning|@testgroup|09:00|早上好\n```\n\n### 某天某个时间给用户发一次消息\n```\nschedmsg addonce note1|123456789|2026-04-21 08:30|记得开会\n```\n\n### 每天凌晨 00:01 给机器人发命令\n```\nschedmsg adddaily money|@bytevirtbot|00:01|/showmethemoney\n```\n\n## 目标写法\n目标支持：\n- `@username`\n- 数字 `chat_id`\n- 群 / 频道用户名\n\n## 目录结构\n```\nschedmsg_plus/\n├── DES.md\n├── README.md\n└── main.py\n```\n\n## 说明\n- 当前轮询检查频率：20 秒\n- 只有真正到点时才发送\n- 已加防重复发送逻辑，避免同一分钟重复触发两次\n
